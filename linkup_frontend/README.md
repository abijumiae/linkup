# LinkUp

## Connect Everything

A worldwide all-in-one social media platform frontend MVP.

### Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

### Pages Included

- `/`
- `/login`
- `/signup`
- `/home`
- `/explore`
- `/messages`
- `/groups`
- `/marketplace`
- `/jobs`
- `/events`
- `/notifications`
- `/settings`
- `/profile`
- `/admin`

### Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

### Important Notes

- Backend is not connected yet.
- All data is dummy data.
- Admin page is for development preview only and will later be protected by authentication and role-based access control.
